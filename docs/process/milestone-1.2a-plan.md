# Milestone 1.2A Plan

## Goal
Scaffold the initial platform foundation using official generators where possible.

## Scope
- NestJS API using Fastify
- React dashboard using Vite
- Chrome extension using Manifest V3
- Shared database package
- Shared UI package

## Approach
- Prefer official generators (Nest CLI, Vite, Prisma, etc.)
- Layer ProcureWiz-specific architecture on top of generated boilerplate
- Keep framework setup minimal and standard

## Verification
- pnpm install
- pnpm build
- pnpm lint
- pnpm test
